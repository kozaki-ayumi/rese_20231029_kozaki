# rese
トップページではお店の一覧が表示される。
ユーザー登録をすることで予約ができ、mypageから予約の確認や来店後のレビューを記載することができる。
またお気に入りの登録もでき、mypageから確認することができる。
管理者画面があり、店舗代表者を登録することができる。
店舗代表者は店舗情報の登録・更新、店舗の予約状況の確認、お支払い、お知らせメールの送信ができる。
また毎朝7時に当日の予約者に確認のメールが届くようになっている。
